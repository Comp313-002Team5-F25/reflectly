Reflectly

Reflectly is a privacy-first, AI-powered reflective conversation web application designed to help users feel heard, gain emotional clarity, and engage in deeper self-reflection through guided, non-clinical dialogue.

Unlike traditional chatbots or wellness apps, Reflectly focuses on active listening: it paraphrases a userâ€™s thoughts to demonstrate understanding, then asks one gentle follow-up question (or offers structured steps when explicitly requested) to encourage meaningful reflection.

This repository serves as both a working product prototype and a case study demonstrating thoughtful UX, accessibility, and ethical AI design.
